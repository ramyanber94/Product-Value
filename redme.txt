################################# ##############################
## Hello there! This is a simple web application that uses
## a machine learning model to predict the price of a car based on
## its features. The app is built using Qwik for the frontend and
## FastAPI for the backend.
## The app uses Prisma as the ORM to connect to a PostgreSQL database
## to store the car data and the predictions.
## The app is designed to be simple and easy to use. The frontend
## is built using Qwik, a new framework for building web applications
## that is designed to be fast and efficient. The backend is built
## using FastAPI, a modern web framework for building APIs with Python.

###################### Steps to run the app ###########################

###################### Frontend ###########################
## 1. Clone the repository
## 2. Install the dependencies using "npm install"
## 3. Create a .env file same as .env.example and add your database connection string
## 4. Run the Prisma migration using "npx prisma migrate dev --name init"
## 5. Run the Prisma generate command using "npx prisma generate"
## 6. Run the app using "npm start"


############################## Backend ###########################
## 1. cd model 
## 2. Install the dependencies using "pip install -r requirements.txt"
## 3. Run the app using "uivicorn model.app:app --reload"
